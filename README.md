## Golden Sun Utility Script

### Game Support

This should currently support the following games:
- Golden Sun (The Broken Seal)
- Golden Sun: The Lost Age

### Emulator Support

This should currently support the following emulators:
- VBA-rr v24
- BizHawk 2.8
